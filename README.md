Copia il codice:
```
git clone https://github.com/paolini/oliquery.git
cd oliquery
```

Prima di iniziare servono le credenziali:
```
export OLI_EMAIL=my-username
export OLI_PASSWORD=my-secret-password
```

Test di funzionamento del login:
```
python api.py
```

Elenco CDRD:
```
python cdrd.py
```

Elenco scuole:
```
python schools.py
```

Elenco distretti:
```
python zones.py
```
